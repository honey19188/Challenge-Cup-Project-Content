# 江汉关·双模智控系统 - 版本对比文档

## 项目概述

江汉关·双模智控系统是一款基于ESP32-S3的智能灯光控制系统，支持温度预设和灯光效果双模式运行。项目经历了**赛时版**和**赛后改良版**两个开发阶段。

---

## 版本对比总览

| 对比项 | 赛时版 | 赛后改良版 |
|--------|--------|------------|
| **开发框架** | MicroPython | Arduino |
| **控制方式** | 语音 + 网页 + 按键 | 语音 + BLE蓝牙 + APP + 按键 |
| **代码结构** | 单文件混合编程 | 模块化分文件管理 |
| **无线通信** | WiFi AP模式 | BLE蓝牙 |
| **移动端** | 网页控制 | 安卓APP |
| **灯光效果** | 8种 | 8种（算法优化） |
| **主要用途** | 比赛展示 | 实用化产品 |

## 软件架构对比

### 赛时版架构（MicroPython单文件）

```
江汉关,txt
├── SSD1306 OLED驱动（内置类）
├── 灯光效果函数（8个）
├── 语音解析函数
├── Web服务器（AsyncIO）
├── MQTT客户端（可选）
└── 主程序（asyncio任务调度）
```

**特点**：单文件结构，代码约770行，使用内嵌式OLED驱动

### 赛后改良版架构（Arduino模块化）

```
固件/main/
├── main.ino          (主程序入口)
├── effects.h          (灯光效果类)
├── su03t.h            (语音模块类)
├── ble_control.h       (蓝牙控制类)
└── oled_display.h     (OLED显示类)
```

**特点**：模块化设计，类封装，BLE特征值通信

## 代码示例对比

### 灯光效果实现

**赛时版（MicroPython）**：
```python
def effect_shengshi(leds, n, t):
    period = 2000
    pos = (t % period) / period * n
    for i in range(n):
        dist = abs(i - pos)
        if dist < 3:
            r, g, b = 255, min(255, int(200 * (1 - dist / 3))), 0
        else:
            r, g, b = 128, 0, 0
        leds[i] = (r, g, b)
```

**赛后改良版（Arduino）**：
```cpp
void effect0() {
    for (int i = 0; i < strip->numPixels(); i++) {
        int brightness = (sin(i * 0.3 + counter * 0.1) + 1) * 127;
        strip->setPixelColor(i, strip->Color(brightness, brightness * 0.3, 0));
    }
    strip->show();
    counter++;
}
```

## 版本选择建议

| 应用场景 | 推荐版本 | 原因 |
|----------|----------|------|
| 比赛展示 | 赛时版 | 网页控制便于演示，代码简单易懂 |
| 产品开发 | 赛后改良版 | 模块化结构易于维护，BLE低功耗 |
| 学习参考 | 赛时版 | MicroPython语法简洁，适合入门 |
| 二次开发 | 赛后改良版 | 类封装完善，易于扩展功能 |

## 技术支持与扩展

- **BLE蓝牙协议**：基于UUID 0000FFE0-0000-1000-8000-00805F9B34FB
- **语音协议**：SU03T UART通信，0xAA 0x55命令格式
- **OLED协议**：I2C地址0x3C/0x3D，128x64分辨率
- **扩展接口**：预留GPIO，可接更多传感器
