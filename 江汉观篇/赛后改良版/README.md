# 赛后改良版 - 江汉关·双模智控系统

## 版本概述

赛后改良版是比赛后针对赛时版进行优化和功能扩展的版本，由MicroPython迁移至Arduino框架开发，实现了更模块化的代码结构和蓝牙低功耗（BLE）控制功能，并配套开发了安卓控制APP。

## 硬件架构

| 组件 | 型号/参数 | 功能 |
|------|----------|------|
| 主控 | ESP32-S3 | 系统核心控制 |
| 灯带 | WS2812B x60 | RGB灯光效果 |
| 语音 | SU03T | 语音命令识别 |
| 传感器 | MQ135 | 空气质量检测 |
| 显示 | SSD1306 I2C OLED 128x64 | 状态信息显示 |
| 通信 | BLE蓝牙 | 无线控制 |

## 功能特性

### 双模式控制
- **温度预设模式**：4档温度（35℃/22℃/10℃/-15℃），对应不同灯光颜色
- **灯光效果模式**：8种动态效果，支持自动巡演

### BLE蓝牙控制（新增）
- 设备名称：ESP32_BLE
- 支持APP连接和控制
- 实时状态推送
- 自动重连机制

### 安卓APP（新增）
- 图形化控制界面
- 模式切换快捷操作
- 状态实时显示
- 蓝牙连接管理

### 灯光效果
| 编号 | 效果名称 | 描述 |
|------|----------|------|
| 0 | 盛世华诞 | 暖黄色正弦波浪效果 |
| 1 | 普天同庆 | 白色渐变呼吸灯 |
| 2 | 渐隐交替 | 彩虹色相交替 |
| 3 | 双色渐变 | 色相连续渐变 |
| 4 | 扫描光带 | 流星划过效果 |
| 5 | 心跳脉冲 | 脉冲紫光效果 |
| 6 | 烟花绽放 | 随机位置烟花 |
| 7 | 拖尾彗尾 | 彩虹拖尾效果 |

## 未来展望

1. **多设备协同** - 支持多台设备组网控制，实现联动效果和分布式展示
2. **云端智能化** - 接入云平台实现数据存储、分析和智能场景推荐
3. **增强现实** - 开发AR功能，通过手机摄像头实现虚实融合的视觉效果
4. **语音AI** - 集成大语言模型，实现自然对话和智能问答
5. **能源管理** - 增加电池管理和太阳能充电方案，实现绿色环保运行

## 改进方向

1. **硬件升级**
   - 增加WiFi+蓝牙双模通信，适应更多应用场景
   - 增加温湿度传感器，实现环境综合监测
   - 增加麦克风模块，支持本地语音唤醒
   - 优化电源设计，支持电池供电实现便携性

2. **软件增强**
   - 迁移回MicroPython以获得更好的开发效率
   - 增加Lua脚本支持，允许用户自定义效果
   - 实现JSON配置文件系统
   - 增加日志系统和远程调试能力

3. **生态扩展**
   - 开发微信小程序控制端
   - 支持Home Assistant集成
   - 开发更多传感器扩展模块
   - 建立开放的特效分享社区

4. **可靠性提升**
   - 增加看门狗定时器防止死机
   - 实现异常捕获和自动恢复
   - 增加固件签名验证确保安全性
   - 优化OTA升级的稳定性
