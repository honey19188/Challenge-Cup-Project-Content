# 赛时版 - 江汉关·双模智控系统

## 版本概述

赛时版是比赛期间使用的版本，基于MicroPython开发，实现了温度预设与灯光效果双模式控制系统。该版本在比赛期间运行稳定完成了各项功能展示。

## 硬件架构

| 组件 | 型号/参数 | 功能 |
|------|----------|------|
| 主控 | ESP32-S3 | 系统核心控制 |
| 灯带 | WS2812B x60 | RGB灯光效果 |
| 语音 | SU03T | 语音命令识别 |
| 传感器 | MQ135 | 空气质量检测 |
| 显示 | SSD1306 I2C OLED 128x64 | 状态信息显示 |

## 功能特性

### 温度预设模式
- 4档温度预设：35℃（炎热）、22℃（温暖）、10℃（凉爽）、-15℃（寒冷）
- 根据温度显示不同颜色灯光（红/黄/绿/蓝）

### 灯光效果模式
1. 盛世华诞 - 红色庆典效果
2. 普天同庆 - 彩虹循环效果
3. 渐隐交替 - 颜色渐变呼吸效果
4. 双色渐变 - 红蓝渐变效果
5. 扫描光带 - 绿色扫描效果
6. 心跳脉冲 - 心跳节奏效果
7. 烟花绽放 - 随机烟花效果
8. 拖尾彗尾 - 彗星拖尾效果

### 控制方式
- **语音控制**：通过SU03T语音模块识别命令
- **网页控制**：WiFi AP模式（192.168.4.1），支持温度切换、效果选择、灯光开关、关机重启
- **按键控制**：BOOT按键单击切换模式，双击播报状态

## 软件架构

```
江汉关,txt (MicroPython)
├── OLED驱动 (SSD1306 I2C)
├── 灯光效果库 (8种效果)
├── 语音解析 (SU03T UART协议)
├── Web服务器 (AsyncIO)
└── 传感器模块 (MQ135)
```

## 未来展望

1. **智能化升级** - 引入AI语音识别，支持自然语言对话和场景智能推荐
2. **物联网拓展** - 支持MQTT协议接入智能家居平台，实现远程控制和数据上报

## 改进方向

1. **硬件优化**

   - 增加更多场景化预设模式
