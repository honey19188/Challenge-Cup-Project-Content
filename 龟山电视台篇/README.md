# 龟山电视塔灯光秀

## 项目概述

这是一个**挑战杯参赛项目**，旨在为龟山电视塔设计一套智能灯光控制系统，支持多种灯光效果、远程控制、语音识别和环境监测功能。

## 核心功能

| 功能 | 说明 |
|------|------|
| **6种灯光模式** | 关闭/真塔秀/节日秀/彩虹流/呼吸变/警示闪 |
| **BLE蓝牙控制** | 通过Android App无线控制 |
| **WiFi网页控制** | 移动端浏览器直接访问 |
| **语音识别控制** | 支持语音指令切换灯光模式 |
| **窗帘电机控制** | 通过语音或网页控制 |
| **OLED显示** | 显示温度/湿度/空气质量 |
| **环境监测** | 温湿度、空气质量传感器 |
| **空气质量报警** | 超标时自动红色警示并语音播报 |
| **自动模式循环** | 每15秒自动切换不同灯光效果 |
| **物理按键控制** | 支持按键切换模式 |

## 软件架构

### 赛后改良版 (BLE版本) - Arduino IDE 开发
- **固件**：`esp32_ble_led.ino` (ESP32 Arduino C++)
- **开发环境**：Arduino IDE + ESP32 开发板支持
- **通信协议**：BLE蓝牙 UART (Nordic NUS)
- **控制端**：Android App (APK)

### 赛时版 (WiFi版本) - MicroPython 开发
- **固件**：`WIFI多功能版.txt` (MicroPython)
- **开发环境**：Thonny IDE
- **通信协议**：WiFi热点 + HTTP Web服务器
- **控制端**：移动端浏览器

## 灯光效果详解

### 1. 真塔秀 (MODE_REAL)
模拟真实电视塔灯光效果：
- 底座：暖白色常亮
- 塔身：水波纹效果向上流动
- 塔尖：红色闪烁

### 2. 节日秀 (MODE_FESTIVAL)
- 底座：暖黄色常亮
- 塔身：彩虹色流动渐变
- 塔尖：红色闪烁

### 3. 彩虹流 (MODE_RAINBOW)
全灯带彩虹渐变效果，颜色平滑过渡

### 4. 呼吸变 (MODE_BREATH)
柔和蓝白色呼吸渐变，亮度周期性变化

### 5. 警示闪 (MODE_TOP_FLASH)
- 底座：白色常亮
- 塔身：熄灭
- 塔尖：红色快速闪烁

### 6. 关闭模式 (MODE_OFF)
全部熄灭

## 控制指令

### BLE指令 (十六进制)
| 指令 | 功能 |
|------|------|
| 0x00 | 关闭 |
| 0x01 | 真塔秀 |
| 0x02 | 节日秀 |
| 0x03 | 彩虹流 |
| 0x04 | 呼吸变 |
| 0x05 | 警示闪 |
| 0xF0 | 窗帘打开 |
| 0xF1 | 窗帘关闭 |

### 文本指令
- `off` - 关闭
- `real` - 真塔秀
- `festival` - 节日秀
- `rainbow` - 彩虹流
- `breath` - 呼吸变
- `top` - 警示闪
- `white/red/blue/warm` - 纯色模式
