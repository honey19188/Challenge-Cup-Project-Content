## 灯光效果详解

### 1. 真塔秀 (MODE_REAL)
模拟真实电视塔灯光效果：
- 底座：暖白色常亮
- 塔身：水波纹效果向上流动
- 塔尖：红色闪烁

### 2. 节日秀 (MODE_FESTIVAL)
- 底座：暖黄色常亮
- 塔身：彩虹色流动渐变
- 塔尖：红色闪烁

### 3. 彩虹流 (MODE_RAINBOW)
全灯带彩虹渐变效果，颜色平滑过渡

### 4. 呼吸变 (MODE_BREATH)
柔和蓝白色呼吸渐变，亮度周期性变化

### 5. 警示闪 (MODE_TOP_FLASH)
- 底座：白色常亮
- 塔身：熄灭
- 塔尖：红色快速闪烁

### 6. 关闭模式 (MODE_OFF)
全部熄灭
