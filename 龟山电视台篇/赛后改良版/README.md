## Android App

应用名称：`龟山电视台灯光蓝牙BEL控制器`

### 功能

- BLE 设备扫描与连接
- 灯光模式切换（6种）
- 纯色模式控制
- 窗帘控制
- 传感器状态查询

---

## 技术实现

### BLE 回调机制

- `MyServerCB`: 连接/断开回调，自动重连广播
- `MyRXCB`: 接收数据回调，保存到 rxValue

### 指令处理

loop 中统一处理 BLE 和语音接收的指令，避免在回调中直接操作硬件。

### 灯光渲染

根据当前模式调用对应渲染函数，使用 Adafruit_NeoPixel 库控制 WS2812B 灯带。
