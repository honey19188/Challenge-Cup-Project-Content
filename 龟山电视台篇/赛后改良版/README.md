# 龟山电视塔灯光秀 - 赛后改良版

比赛后优化的版本，基于 **Arduino IDE** 开发，采用 **BLE蓝牙** 通信。

---

## 硬件平台

| 组件 | 型号 |
|------|------|
| 主控芯片 | ESP32-WROOM-32E |
| LED灯带 | WS2812B (64个RGB LED) |
| 语音模块 | 串口语音识别模块 |

---

## 引脚定义

| 功能 | 引脚 |
|------|------|
| LED灯带 | GPIO 25 |
| 语音RX | GPIO 17 |
| 语音TX | GPIO 16 |

---

## 灯光效果

| 模式 | 说明 |
|------|------|
| MODE_OFF | 关闭 |
| MODE_REAL | 真塔秀 - 模拟真实电视塔灯光 |
| MODE_FESTIVAL | 节日秀 - 彩虹流动 |
| MODE_RAINBOW | 彩虹流 |
| MODE_BREATH | 呼吸变 |
| MODE_TOP_FLASH | 警示闪 - 红色塔尖闪烁 |

---

## 控制指令

### BLE十六进制指令

| 指令 | 功能 |
|------|------|
| 0x00 | 关闭 |
| 0x01 | 真塔秀 |
| 0x02 | 节日秀 |
| 0x03 | 彩虹流 |
| 0x04 | 呼吸变 |
| 0x05 | 警示闪 |
| 0xF0 | 窗帘打开 |
| 0xF1 | 窗帘关闭 |
| 0x50 | 查询状态 |

### 文本指令

| 指令 | 功能 |
|------|------|
| off | 关闭 |
| real | 真塔秀 |
| festival | 节日秀 |
| rainbow | 彩虹流 |
| breath | 呼吸变 |
| top | 警示闪 |
| white/red/blue/warm | 纯色模式 |

---

## 控制方式

### Android App
通过蓝牙连接，控制灯光模式、窗帘开关等

### 语音控制
支持语音指令切换灯光模式

---

## 开发环境

- **语言**：Arduino C++
- **IDE**：Arduino IDE + ESP32 开发板支持
- **通信**：BLE蓝牙 UART (Nordic NUS)

## 后期可扩展功能

当前版本采用模块化设计，具备以下扩展能力：

### 硬件扩展
| 功能 | 说明 | 实现方式 |
|------|------|----------|
| OLED显示 | 128x64 I2C OLED | GPIO 21(SDA)/22(SCL) |
| 温湿度传感器 | DHT11/DHT22 | 单线数字接口 |
| 空气质量传感器 | PM2.5/TVOC | ADC或I2C接口 |
| 窗帘步进电机 | 4线步进电机 | GPIO 12-15 |

### 软件扩展
| 功能 | 说明 |
|------|------|
| WiFi双模 | 同时支持BLE+WiFi |
| 远程OTA升级 |通过网络更新固件 |
| 多设备联动 | 主从模式控制多个灯塔 |
| 环境数据上报 | MQTT对接云平台 |
| 定时任务 | 自动开关灯光 |

### 预留引脚
```
GPIO 21, 22  - I2C总线 (OLED/传感器)
GPIO 12-15   - 步进电机控制
GPIO 34-39   - ADC采集 (空气质量)
GPIO 26, 27  - 预留IO
```
