# 龟山电视塔灯光秀 - 赛时版

比赛期间使用的版本，基于 **MicroPython** 开发。

---

## 硬件平台

| 组件 | 型号 |
|------|------|
| 主控芯片 | ESP32 |
| LED灯带 | WS2812B (60个RGB LED) |
| 语音模块 | 串口语音识别模块 |
| 显示模块 | SSD1306 OLED (128x64) |
| 传感器 | 温湿度一体传感器 |
| 电机驱动 | 步进电机(4线) |

---

## 引脚定义

| 功能 | 引脚 |
|------|------|
| LED灯带 | GPIO 18 |
| 物理按键 | GPIO 0 |
| 温度按键S1(红) | GPIO 3 (35℃) |
| 温度按键S2(黄) | GPIO 8 (22℃) |
| 温度按键S3(绿) | GPIO 6 (10℃) |
| 温度按键S4(蓝) | GPIO 7 (-15℃) |
| OLED时钟 | GPIO 41 |
| OLED数据 | GPIO 42 |
| 语音RX | GPIO 10 |
| 语音TX | GPIO 9 |
| TTS_TX | GPIO 17 |
| TTS_RX | GPIO 16 |
| 步进电机 | GPIO 38, 39, 40, 2 |

---

## 灯光效果

| 模式 | 说明 |
|------|------|
| MODE_OFF | 关闭 |
| MODE_REAL | 真塔秀 - 模拟真实电视塔灯光 |
| MODE_FESTIVAL | 节日秀 - 彩虹流动 |
| MODE_RAINBOW | 彩虹流 |
| MODE_BREATH | 呼吸变 |
| MODE_TOP_FLASH | 警示闪 - 红色塔尖闪烁 |

## 控制方式

### WiFi网页控制
- AP热点名：`龟山电视塔`
- 密码：`12345678`
- 网页地址：`http://192.168.4.1`

### 语音控制
支持语音指令切换灯光模式

### 物理按键
- 单击：切换灯光模式
- 双击：广播温湿度信息
- 长按3秒：电机旋转720度

---

## 特色功能

- **OLED显示**：实时显示温度/湿度/空气质量
- **空气质量报警**：超标时红色警示+语音播报
- **温湿度联动**：根据环境自动调整灯光模式
- **窗帘控制**：语音或网页控制步进电机
- **自动循环模式**：每15秒自动切换灯光效果
