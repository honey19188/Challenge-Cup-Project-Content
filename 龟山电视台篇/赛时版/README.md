## 功能特性

### 灯光模式

| 模式 | 说明 |
|------|------|
| mode 0 | 关闭 |
| mode 1 | 单点流动 (金橙红) |
| mode 2 | 彩虹渐变 |
| mode 3 | 七彩闪烁 |
| mode 4 | 蓝金波动 |
| mode 5 | 五彩分段 |

### 控制方式

1. **WiFi 网页控制** - 连接热点后通过浏览器访问
2. **物理按键控制** - 短按切换模式，长按 2 秒进入巡演模式
3. **SU-03T 语音控制** - 通过 SU-03T 语音识别模块控制
4. **自动循环模式** - 每 15 秒自动切换不同效果 (巡演模式)

### 智能功能

- **自动循环模式**：巡演模式下每 15 秒自动切换不同动画
- **按键防抖**：按键消抖处理，防止误触发
- **深度睡眠关机**：支持网页关机进入深度睡眠模式
- **内存管理**：定期垃圾回收，保证运行稳定

## 技术实现

### 灯光效果算法

| 模式 | 算法描述 |
|------|----------|
| anim_1 | 单点流动：金色亮点沿灯带移动，带橙红色拖尾 |
| anim_2 | 彩虹渐变：HSV 色彩空间连续过渡 |
| anim_3 | 七彩闪烁：基于素数原理的伪随机多彩闪烁 |
| anim_4 | 蓝金波动：正弦波驱动的蓝金色交替 |
| anim_5 | 五彩分段：5 种颜色等间隔分布 |
| anim_6 | 渐变拖尾：亮度递减的彩色拖尾效果 |
| anim_7 | RGB 波浪：三通道正弦波相位偏移 |
| anim_8 | 往复运动：彩点在灯带两端往复移动 |
| anim_9 | 颜色索引：基于正弦值的颜色查找表 |
| anim_10 | 定时换色：10 秒切换一种纯色 |

### 按键处理

- **短按 (< 2秒)**：切换下一个动画模式
- **长按 (> 2秒)**：进入/退出巡演模式

### 语音指令

| 指令 | 功能 |
|------|------|
| 0x00 | 关闭 |
| 0x01~0x0A | 对应模式 1-10 |
| 0xFE | 进入巡演模式 |
